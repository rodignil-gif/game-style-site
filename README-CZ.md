# Vývoj stylu her

Tento web je věnován vývoju stylu ve videohrách. Zde najdete základy vytváření jedinečného vizuálního stylu pro vaše hry.

## Popis

Web obsahuje informace o:
- Vizuálním designu
- Barevných paletách
- Typografii
- Atmosféře her

## Technologie

- HTML5
- CSS3 (Bootstrap)
- JavaScript

## Instalace

1. Naklonujte repozitář.
2. Otevřte index.html v prohlížeči.
