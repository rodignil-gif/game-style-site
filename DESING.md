# Bunkr 9: Biokov - Design Document

**Název:** Bunkr 9: Biokov  
**Koncept:** 2D platformer roguelike v podzemním biomechanickém bunkru. Po biotechnologické katastrofě se komplex stává živým organizmem.

- Postavy: Rian (kurýr), Doktorka Sena, K-0B (dron)
- Nepřátelé: Maso-strojní strážcové, turella, infikovaní technici
- Bossové: Srdce Reaktoru, Správce Šachty, Matrice Kostí
- Barvy: Tmavě šedá, kyselkově zelená neon, rezavě rudá
